Introduction

This is a tutorial for more advanced web scraping of JavaScript elements. I will use the same example as in my beginner web scraping tutorial. In that example, I was scraping price and product information from Walmart.com for Kraft Macaroni and Cheese.1

This is because the HTML contains a lot of extra information that is not visible on the page and most of it was contained within JavaScript. The JavaScript contained a lot of fascinating extra information including the following:

  • Product name
  • UPC code
  • Seller’s name and ID
  • Retail price
  • Sale price
  • In-store price
  • Quantity available (inventory)

From an economic perspective, these extra fields are invaluable (especially the UPC code). The inventory count is also fascinating.
